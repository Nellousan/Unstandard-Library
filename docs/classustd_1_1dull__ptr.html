Detailed Description

template<class T>
class ustd::dull_ptr< T >

An upgraded std::weak_ptr easier to use.

The main features of this class is the ability to access the stored pointer easily, as well as an easier way to tell if two pointer are the same or to tell if a pointer has expired by comparing it with nullptr.
